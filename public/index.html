Text-based Game - Incremental Development

This project demonstrates the step-by-step development of a text-based game. Each part builds upon the previous one.

  • Part 1: Basic Input/Output - Enter text and display it in a paragraph
  • Part 2: Input Analysis - Process input commands and respond accordingly
  • Part 3: Player Class & Challenges - Introduce the Player class with developer challenges
  • Part 4: Gemini AI Integration - Send prompts to Gemini AI and display responses
  • Part 5: System Prompts & Multichat - Add system prompt selection and multichat toggle
  • Part 6: JSON Story Adventure - Parse JSON responses for choose-your-own-adventure gameplay
  • Part 7: Guidable Story Engine - Interactive fiction with Story Bible, State, and AI-powered narrative
